# Contributing to Codex

Thank you for your interest in contributing to Codex.

Codex is the design system for Wikimedia and is maintained by teams in the Product & Technology Department while stewarded by the [Codex Steering Committee](https://www.mediawiki.org/wiki/Codex/Steering_Committee) of the [Wikimedia Foundation](https://wikimediafoundation.org/).

## Support level

Codex is under active maintenance.

## How to contribute

Before contributing, please review the [Codex contributing documentation](https://doc.wikimedia.org/codex/latest/contributing/overview.html).

## Code contributions

The canonical repository is on [Gerrit](https://gerrit.wikimedia.org/r/admin/repos/design/codex).

## Reporting issues

Tasks and bug reports are tracked on [Phabricator](https://phabricator.wikimedia.org/tag/codex/). Please search for existing tasks before creating a new one.

## Code of Conduct

Participation in Wikimedia technical spaces is governed by the [Wikimedia Code of Conduct](https://www.mediawiki.org/wiki/Special:MyLanguage/Code_of_Conduct). See also [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) in this repository.
